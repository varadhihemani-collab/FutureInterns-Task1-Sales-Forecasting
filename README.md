# Task 1: Sales & Demand Forecasting

## Description
This project focuses on analyzing historical sales data to forecast future demand using time series forecasting techniques. It helps businesses improve inventory planning and make better business decisions.

## Project Files

| File | Purpose |
|------|---------|
| Sales_Forecasting.ipynb | Main notebook containing data cleaning, EDA, model building, and forecasting |
| Project_Report.docx | Detailed project report with results, graphs, and conclusions |

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Prophet / ARIMA

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Time Series Forecasting
5. Model Evaluation
6. Future Sales Prediction

## How to Run

```bash
git clone https://github.com/varadhihemani-collab/FutureInterns-Task1-Sales-Forecasting.git
```

- Open the project folder.
- Install the required libraries.
- Run `Sales_Forecasting.ipynb` using Jupyter Notebook or Google Colab.

## Results
The model forecasts future sales trends to support demand forecasting and inventory management.
