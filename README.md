# Task 1: Sales & Demand Forecasting

## Description
This project analyzes historical sales data to forecast future demand. 
It helps in inventory planning and business decision making using time series forecasting.

## Project Files
| File | Purpose |
| --- | --- |
| `Sales_Forecasting.ipynb` | Main notebook with data cleaning, EDA, modeling and forecasting |
| `Project_Report.docx` | Detailed report with results, graphs and conclusions |

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn 
- Scikit-learn
- Prophet / ARIMA for time series forecasting
